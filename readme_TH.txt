SHIPPOP เป็น Plugin สำหรับ WooCommerce และเป็น Plugin Offical จาก [SHIPPOP](https://shippop.com) ที่จะช่วยให้คุณสามารถจัดการเกี่ยวกับการจัดส่งของผ่านระบบของ SHIPPOP ได้อย่างง่ายดาย

== รายละเอียด ==

รายละเอียดความสามารถของ Plugin นี้ที่จะช่วยคุณในการจัดการเกี่ยวกับการจัดส่งสินค้า

= ส่วนของ เจ้าของร้านค้า =

*   กำหนดข้อมูล ชื่อ ที่อยู่ เบอร์ติดต่อ สำหรับใบปะหน้า และ ข้อมูล ที่อยู่ใบแจ้งหนี้และใบเสร็จได้
*   กำหนดขนาดกล่องที่ใช้จัดส่งได้ (optinal)
*   อัพเดตสถานะการขนส่งแบบ Realtime
*   เลือกขนส่งที่ต้องการ พร้อมคำนวณราคาขนส่ง โดยประมาณให้
*   แสดงรายการจัดส่งพัสดุ และ รายงาน COD
*   สามารถออกรายงานเป็น .csv ได้
*   ระบบแสดงเลข Tracking ให้ติดตามสถานะการจัดส่งทั้งในหน้า รายละเอียดคำสั่งซื้อ และ หน้า รายการจัดส่ง
*   สามารถปริ้นใบปะหน้าได้หลายขนาด
*   ปรับสถานะของคำสั่งซื้อเป็น สำเร็จ เมื่อ พัสดุ ได้ส่งถึงที่หมายได้อย่างถูกต้อง

= ส่วนของ ลูกค้า =

*   สามารถติดตามพัสดุ ได้จากหน้า รายการคำสั่งซื้อ

== Installation ==

= ความต้องการขั้นต่ำ =

* PHP 5.5 ขึ้นไป
* WooCommerce 3.1.2 ขึ้นไป

= วิธีการแนะนำ =

ติดตั้งและใช้งาน Plugin จากหน้า "Plugins > Add New"

= วิธีการติดตั้งแบบ manual =

1. แตกไฟล์ไปที่ โฟลเดอร์ /wp-content/plugins
2. เปิดใช้งาน Plugin ในหน้า \wp-admin\plugins.php
